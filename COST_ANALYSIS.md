# ניתוח עלויות Cookit

## סיכום עלויות API

### Serper.dev (חיפוש Google + תמונות)
- **Free Tier**: 2,500 credits/month
- **Paid**: $0.30-$1.00 per 1,000 credits
- **חיפוש רגיל**: 1 credit
- **חיפוש תמונות**: 1 credit

### Google Gemini API (חילוץ + תרגום)
- **Gemini 3.1 Flash-Lite**: $0.25/1M input + $1.50/1M output
- **Gemini 2.5 Flash-Lite**: $0.10/1M input + $0.40/1M output (מומלץ!)
- **Free Tier**: 1,000 requests/day

## חישוב עלות לחיפוש אחד

```
עלות חיפוש ממוצע:
├── 4 חיפושי Serper (2 עברית + 2 אנגלית)     = 4 credits  ≈ $0.0004
├── 5 תמונות מרכיבים (Serper)                = 5 credits  ≈ $0.0005
├── חילוץ מתכון (Gemini ~1,000 tokens)       =            ≈ $0.0002
└── תרגום כותרות (batch, מתפצל לכל החיפוש)   =            ≈ $0.0001

סה"כ לחיפוש: ~$0.0012 (כ-1.2 אגורות)
```

## המלצות להוזלת עלויות

1. **שימוש ב-Gemini 2.5 Flash-Lite** במקום 3.1 (חיסכון של 60%)
2. **קאשינג אגרסיבי** - שמירת תוצאות חיפוש ל-24 שעות
3. **Batch translation** - תרגום כל הכותרות בבת אחת
4. **הגבלת תמונות** - טעינה עצלה של תמונות מרכיבים
5. **Fallback ל-DuckDuckGo** כש-Serper נגמר
