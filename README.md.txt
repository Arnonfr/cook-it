# Cook.it

תוסף כרום לחילוץ מתכונים מאתרי בישול.

## התקנה
1. הורד את הקוד
2. פתח chrome://extensions
3. הפעל מצב מפתח
4. טען תיקייה לא דחוסה

## תכונות
- חילוץ מקומי מאתר Foody
- חילוץ באמצעות AI מכל אתר
- תצוגה נקייה
- שינוי גודל טקסט

## טכנולוגיות
- JavaScript
- Chrome Extensions API
- Google AI API
