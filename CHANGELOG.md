# Changelog - CookIt

כל השינויים הבולטים בפרויקט יתועדו כאן.

## [1.0.3] - 2025-03-22

### שיפורי אבטחה
- הוספת Rate Limiting (50 בקשות/15 דקות, 20 parses/שעה)
- הוספת Helmet ל-security headers
- הגבלת CORS ב-production
- מסכת מפתחות API (masking) בהגדרות

### שיפורי פונקציונליות
- שיפור חילוץ שלבי מתכון - פיצול אגרסיבי לטקסט ארוך
- תיקון תצוגת מרכיבים בתוצאות חיפוש (ingredients preview)
- הוספת ערוץ התראות לשיתוף מתכונים

## [1.0.2] - 2025-03-22

### תיקונים
- תיקון state management בהגדרות פרופיל
- שמירת העדפות משתמש ב-localStorage
- תיקון שימוש ב-Serper API לחיפוש

## [1.0.1] - 2025-03-21

### תיקונים
- תיקון כפתורי פרופיל - שימוש ב-React state
- תיקון חיפוש - שימוש ב-API אמיתי
- תיקון שיתוף - שימוש בהתראות במקום overlay

## [1.0.0] - 2025-03-21

### שחרור ראשוני
- מסך בית עם קטגוריות ומתכונים מהקהילה
- חיפוש מתכונים (מקומי + web)
- צפייה בפרטי מתכון עם מרכיבים ושלבי הכנה
- מסך פרופיל עם הגדרות תזונתיות
- שמירת מתכונים לספרייה האישית
- שיתוף מתכונים מדפדפן לאפליקציה

### ידוע
- אייקון האפליקציה הוא ברירת מחדל של Capacitor (צריך להחליף)
- ה-backend צריך להיות בענן לשימוש בטלפון אמיתי
